확정적으로 읽지 않는 것이 맞습니다.\n\n셋째, 동남아의 점유 0.0000은 '구찌가 그 시장에서 존재감이 없다'는 신호가 아닙니다. 오늘 표본에서 대칭 비교가 가능한 경쟁 브랜드 언급을 확보하지 못해 비율 계산이 0이 된 데이터 얇음의 결과입니다. 표본이 채워지면 다시 정상 범위로 돌아올 가능성이 높습니다.\n\n넷째, 동북아와 동남아 근거에는 신선도
